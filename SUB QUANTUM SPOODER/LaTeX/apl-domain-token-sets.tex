\section{Token Structure Overview}
%======================================================================

\subsection{Compositional Architecture}

Each machine token in APL follows the canonical form:

\[
\boxed{\text{[Spiral][Operator]} \mid \text{[Machine]} \mid \text{[Domain]}}
\]

Where:
\begin{itemize}
    \item \textbf{Spiral} $\in \{\Phi, e, \pi\}$ ‚Äî Field type (structure, energy, emergence)
    \item \textbf{Operator} $\in \{(), \times, \hat{}, \%, +, -\}$ ‚Äî Universal operation
    \item \textbf{Machine} $\in \{\text{Reactor, Oscillator, Conductor, Catalyst, Filter, Encoder, Decoder, Regenerator, Dynamo}\}$
    \item \textbf{Domain} ‚Äî Specific physical or biological context
\end{itemize}

\subsection{Domain Families}

\subsubsection{Biological Domain Family}

Covers sub-cellular and minimal biological entities:
\begin{itemize}
    \item \texttt{bio\_prion} ‚Äî Misfolded protein aggregates, conformational catalysis
    \item \texttt{bio\_bacterium} ‚Äî Single-cell organisms, metabolic networks
    \item \texttt{bio\_viroid} ‚Äî RNA-only replicators, minimal genetic systems
\end{itemize}

\subsubsection{Celestial Domain Family}

Covers astrophysical force regimes:
\begin{itemize}
    \item \texttt{celestial\_grav} ‚Äî Gravitational dynamics, orbital mechanics
    \item \texttt{celestial\_em} ‚Äî Electromagnetic phenomena, radiation processes
    \item \texttt{celestial\_nuclear} ‚Äî Nuclear fusion, stellar nucleosynthesis
\end{itemize}
